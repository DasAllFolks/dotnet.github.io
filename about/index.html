About

This site displays projects that are part of the .NET Foundation to make them easy to discover and to broaden the set of contributors. The projects are listed in ‘Repo Rank’ order.

Repo card legend

 Stars A combination of a 'like' and a bookmark that developers can apply to a repo.
Forks A copy of a repo that enables you 100% control of the code, including merging back your changes.
Contributors The distinct set of developers that have committed to a repo.
Open issues The set of bugs and feature requests for a repo.
Repo rank A closed-set scoring system based on the "OSS Awesomeness" algorithm.
